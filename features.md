# Features in Split Smart

### ***Below are the main features that will be in the split smart :-***

- **Create User** --> as it says create/registers the user.
- **log in** --> lets the user log back in with authorization.

- **Create Groups** --> creates groups for friends and participants.

- **Send invitation** --> create a group invitation link which could be send to required members of the groups.

- **Join group** --> when accesing through the invitation link user is joined into the group upon confirmation

- **Log expense** --> used to log the expense within the group , split type and the payer too.

- **View Balance** --> used to view user's current balance (i.e : what you owe to whom or who owe to you).

- **Pay Back**  --> used before settling the balance by paying back to the payee.

- **Mark Settle** --> used to mark the transactions as settled upon complete payback.
